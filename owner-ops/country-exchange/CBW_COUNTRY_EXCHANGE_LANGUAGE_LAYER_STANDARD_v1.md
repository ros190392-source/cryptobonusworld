# CBW Country × Exchange Language Layer Standard v1

## Rule

One canonical fact layer supports multiple language layers.

```text
Canonical research truth
→ Russian
→ Kazakh
→ English
```

Facts, dates, fees, availability, confidence, sources and restrictions remain identical.

Language adaptation may change:

- phrasing;
- headings;
- examples;
- terminology;
- FAQ wording;
- title/meta;
- internal-link anchors;
- date/currency formatting.

It may not:

- change availability;
- strengthen or weaken legal conclusions;
- invent local bonuses;
- omit limitations;
- change checked dates;
- alter ranking position;
- remove source references.

## Kazakhstan example

Supported languages:

- Russian (`ru`)
- Kazakh (`kk`)
- English (`en`)

Recommended page titles:

- RU: `Bybit в Казахстане: доступность, P2P, KZT, комиссии и промокод`
- KK: `Қазақстандағы Bybit: қолжетімділік, P2P, KZT, комиссиялар және промокод`
- EN: `Bybit in Kazakhstan: availability, P2P, KZT, fees and promo code`

## Workflow

```text
approved evidence package
→ language brief
→ adaptation
→ glossary validation
→ factual parity validation
→ local editorial review
→ SEO/AI-answer review
→ preview
→ owner approval
```
