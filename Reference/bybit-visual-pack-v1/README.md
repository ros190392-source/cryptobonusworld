## Evergreen rule

These images must not include changing data:

- no promo code
- no bonus amount
- no date
- no "verified June 2026"
- no temporary offer conditions

Changing data must live in HTML/text/cards, not inside images.

---

## Approved visual wording

**Allowed:**
- BYBIT
- REFERRAL CODE
- CLAIM BONUS

**Not allowed:**
- CRYPTOBONUSW
- Up to $30,000
- June 2026
- checked date
- temporary bonus conditions
