# AI System

Infrastructure for AI-operated editorial, monitoring, and automation workflows.

## Contents
- `agents/` — Agent definitions and capabilities
- `prompts/` — Reusable prompts for editorial and verification tasks
- `workflows/` — Workflow definitions and sequencing
- `queues/` — Approval queues, refresh queues, pending work
- `reports/` — Archived reports and audit snapshots
- `memory/` — Context transfer documents for new AI sessions

## Quick Start
For a new AI session: read `memory/QUICKSTART_AI_CONTEXT.md` first.
For full context: read `memory/AI_CONTEXT_TRANSFER.md`.
